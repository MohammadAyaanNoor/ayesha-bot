project overview:-

this project is a task given by a company named Ayesha bot as a test for the candidate applying for the position of a threejs developer.
In this project , a 3d plane is created and a 3d humanoid model is imported and animation is applied to it.
there are user interation as well such as on click the animation stops and then on again click the animation starts. 
by pressing the key 'C' on keyboard,the user can change the camera angles.

tech stack:-
Languages : HTML,CSS,JavaScript
Libraries : Three.js
Tools : Vite

Setup guide :

Open the given files in vs code and run 'npm i' on the terminal to install the node modules and then run 'npm run dev' to start the server.
